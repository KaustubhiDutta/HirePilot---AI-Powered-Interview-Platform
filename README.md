## 📌 Overview

**HirePilot** is an AI-powered mock interview platform that helps candidates prepare for technical interviews through adaptive AI-driven interview simulations. It provides realistic interview experiences with voice interaction, real-time feedback, and comprehensive performance reports.


---

## ✨ Features

### 🤖 AI-Powered Interviews
- Realistic voice-based interview simulations
- Text-to-Speech for questions (female/male voices)
- Speech-to-Text for candidate answers
- Real-time answer evaluation and feedback

### 👥 Multiple AI Interviewers
| Name | Voice | Role | Style |
|------|-------|------|-------|
| Tanya | Female | Senior Software Engineer | Friendly & Detailed |
| Isha | Female | Technical Recruiter | Professional & Thorough |
| Rohan | Male | Engineering Manager | Challenging & Insightful |

### 📝 Role-Specific Questions
Tailored questions for 7+ roles:
- Software Engineer
- Frontend Developer
- Backend Developer
- Full Stack Developer
- Data Analyst
- AI/ML Engineer
- DevOps Engineer

### 📊 Real-time Evaluation (6 metrics)
| Metric | Range | Description |
|--------|-------|-------------|
| Accuracy | 0-10 | Technical correctness |
| Clarity | 0-10 | How well explained |
| Relevance | 0-10 | Answers the question |
| Depth | 0-10 | Detail and examples |
| Communication | 0-10 | Structure and language |
| Time Efficiency | 0-10 | Concise but complete |

### 📈 Adaptive Difficulty
- Easy → Medium → Hard based on performance
- Real-time difficulty adjustment
- Early termination for very low performance

### 📄 Resume & JD Analysis
- Upload PDF/DOCX resumes for personalized questions
- Upload or paste Job Descriptions
- AI analyzes and asks role-specific questions

### 🎯 Communication Practice Module
- Voice-based practice sessions
- Detects filler words (um, uh, like)
- Analyzes answer structure and examples
- Provides improvement roadmap

### 📑 Comprehensive Reports
- Overall Readiness Score (0-100)
- Strong / Average / Needs Improvement categories
- Strengths and weaknesses analysis
- Actionable improvement plan
- Downloadable JSON report

### 🔐 User Authentication
- Email/Password signup/login
- Google Sign-In integration
- Protected routes for authenticated users
